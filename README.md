#  *Experiment 6: Determine experimentally the parameter of Two Port Networks*

---

## *Aim*
To determine *Z, **Y, **h, and **ABCD* parameters of *single* and *cascaded* two-port networks experimentally and verify their interrelationships.

---

##  *Procedure*

### *Step 1 — Enter Circuit Values*
- Enter the *supply voltage* and *frequency*.  
- Enter *R, **L, **C* values for the M or N network.  
- Select network type: *M, **N, or **MN (cascaded)*.

---

##  *Parameter Selection Method*
Under *Controls*, choose one parameter group:  
- *Z*  
- *Y*  
- *h*  
- *ABCD*  

Corresponding *instruction box* appears → Follow steps to obtain each parameter.

---

#  *1. Z–Parameter Calculation*

### *Case 1 — Obtain Z₁₁ and Z₂₁*
1. Apply power at *Input Port* and set *Output Port = Intermediate*.  
2. Switch ON supply → Click *Simulate*.  
3. Measure:  
   - Z₁₁ = V₁ / I₁ (I₂ = 0)  
   - Z₂₁ = V₂ / I₁ (I₂ = 0)  
4. Switch OFF supply.

### *Case 2 — Obtain Z₁₂ and Z₂₂*
1. Apply power at *Output Port, set **Input Port = Intermediate*.  
2. Switch ON → Click *Simulate*.  
3. Measure:  
   - Z₁₂ = V₁ / I₂ (I₁ = 0)  
   - Z₂₂ = V₂ / I₂ (I₁ = 0)  
4. Switch OFF supply.

---

#  *2. Y–Parameter, h–Parameter, ABCD–Parameter*
Repeat the same method:  
- Choose *desired tab* (M / N / MN).  
- Select *Y, **h, or **ABCD*.  
- Follow the instruction box to obtain:  
  - Y₁₁, Y₁₂, Y₂₁, Y₂₂  
  - h₁₁, h₁₂, h₂₁, h₂₂  
  - A, B, C, D
